# WebDev-FE

#### User manual

1. Install node
    ````
    https://nodejs.org/en/download/
    ````
   
2. Install angular
    ````
    npm install -g @angular/cli
    ````

3. Install yarn
    ```
    npm install -g yarn
    ```

4. Build project
    ```
    yarn install
    ```

5. Run project
    ```
    ng serve
    ```
6. Error fix
    ```
    export NODE_OPTIONS=--openssl-legacy-provider
    ```
